title: "Quickstart"
description: "Build a working Go HTTP server with gorilla/mux in under 5 minutes."
---

<Info>
  **Prerequisites**: Go 1.20 or higher. Run `go version` to check.
</Info>

<Steps>

<Step title="Install gorilla/mux">

In your module directory, add `gorilla/mux` as a dependency:

```bash
go get -u github.com/gorilla/mux
```

</Step>

<Step title="Create main.go">

Create a file called `main.go` with the following content. It registers three routes and starts an HTTP server with sensible timeouts:

```go
package main

import (
    "encoding/json"
    "log"
    "net/http"
    "time"

    "github.com/gorilla/mux"
)

func main() {
    r := mux.NewRouter()

    r.HandleFunc("/", homeHandler).Methods("GET")
    r.HandleFunc("/products/{id}", getProductHandler).Methods("GET")
    r.HandleFunc("/products", createProductHandler).Methods("POST")

    srv := &http.Server{
        Handler:      r,
        Addr:         ":8080",
        WriteTimeout: 15 * time.Second,
        ReadTimeout:  15 * time.Second,
    }

    log.Println("Listening on :8080")
    log.Fatal(srv.ListenAndServe())
}

func homeHandler(w http.ResponseWriter, r *http.Request) {
    w.Header().Set("Content-Type", "application/json")
    json.NewEncoder(w).Encode(map[string]string{"status": "ok"})
}
```

</Step>

<Step title="Add a route with a path variable">

`gorilla/mux` lets you capture URL segments using `{name}` syntax. Use `mux.Vars(r)` inside the handler to read the captured values:

```go
func getProductHandler(w http.ResponseWriter, r *http.Request) {
    vars := mux.Vars(r)
    id := vars["id"]

    w.Header().Set("Content-Type", "application/json")
    json.NewEncoder(w).Encode(map[string]string{"id": id})
}
```

A request to `GET /products/42` sets `vars["id"]` to `"42"`.

You can also restrict a variable to a specific pattern with `{id:[0-9]+}`, which only matches numeric IDs.

</Step>

<Step title="Add a POST handler">

```go
func createProductHandler(w http.ResponseWriter, r *http.Request) {
    var body map[string]any
    if err := json.NewDecoder(r.Body).Decode(&body); err != nil {
        http.Error(w, "invalid JSON", http.StatusBadRequest)
        return
    }

    w.Header().Set("Content-Type", "application/json")
    w.WriteHeader(http.StatusCreated)
    json.NewEncoder(w).Encode(body)
}
```

</Step>

<Step title="Add middleware">

Middleware in `gorilla/mux` uses the standard `func(http.Handler) http.Handler` signature. Attach it to a router with `r.Use()`:

```go
func loggingMiddleware(next http.Handler) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        log.Printf("%s %s", r.Method, r.RequestURI)
        next.ServeHTTP(w, r)
    })
}
```

Register it before starting the server:

```go
r.Use(loggingMiddleware)
```

Middleware is executed in the order it is added and runs for every matched route.

</Step>

<Step title="Run the server">

```bash
go run main.go
```

The server starts on port 8080. Test your routes:

```bash
curl http://localhost:8080/
curl http://localhost:8080/products/42
curl -X POST http://localhost:8080/products -d '{"name":"widget"}' -H "Content-Type: application/json"
```

</Step>

</Steps>

## Complete example

Here is the full `main.go` combining all of the steps above:

```go
package main

import (
    "encoding/json"
    "log"
    "net/http"
    "time"

    "github.com/gorilla/mux"
)

func loggingMiddleware(next http.Handler) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        log.Printf("%s %s", r.Method, r.RequestURI)
        next.ServeHTTP(w, r)
    })
}

func homeHandler(w http.ResponseWriter, r *http.Request) {
    w.Header().Set("Content-Type", "application/json")
    json.NewEncoder(w).Encode(map[string]string{"status": "ok"})
}

func getProductHandler(w http.ResponseWriter, r *http.Request) {
    vars := mux.Vars(r)
    id := vars["id"]

    w.Header().Set("Content-Type", "application/json")
    json.NewEncoder(w).Encode(map[string]string{"id": id})
}

func createProductHandler(w http.ResponseWriter, r *http.Request) {
    var body map[string]any
    if err := json.NewDecoder(r.Body).Decode(&body); err != nil {
        http.Error(w, "invalid JSON", http.StatusBadRequest)
        return
    }

    w.Header().Set("Content-Type", "application/json")
    w.WriteHeader(http.StatusCreated)
    json.NewEncoder(w).Encode(body)
}

func main() {
    r := mux.NewRouter()

    r.Use(loggingMiddleware)

    r.HandleFunc("/", homeHandler).Methods("GET")
    r.HandleFunc("/products/{id}", getProductHandler).Methods("GET")
    r.HandleFunc("/products", createProductHandler).Methods("POST")

    srv := &http.Server{
        Handler:      r,
        Addr:         ":8080",
        WriteTimeout: 15 * time.Second,
        ReadTimeout:  15 * time.Second,
    }

    log.Println("Listening on :8080")
    log.Fatal(srv.ListenAndServe())
}
```

<Tip>
  Setting `WriteTimeout` and `ReadTimeout` on `http.Server` is important for production servers — it prevents slow clients from holding connections open indefinitely.
</Tip>
